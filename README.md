# snapchat- (One-file)

Repo statique (maquette) basé sur des écrans de connexion Snapchat.

## Démo
Ouvre directement :
- `app.html`

> Les assets images (dossier `img/`) ont été supprimés pour que tout soit contenu dans `app.html`. Si tu veux retrouver le design original avec icônes/eye/back, il faut réajouter `img/` ou intégrer les images en base64.
